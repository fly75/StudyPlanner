# StudyPlanner

Кросплатформений застосунок для планування навчальної діяльності студентів.

## Технології
.NET MAUI 8, C# 12, MVVM (CommunityToolkit), SQLite, BCrypt.Net-Next, 
Plugin.LocalNotification, інтеграція з Claude API (Anthropic)

## Архітектура
MVVM з поділом на View / ViewModel / Services / Data (Repository pattern), 
Dependency Injection через Microsoft.Extensions.DI

## Запуск
1. Відкрити `MauiApp16.sln` у Visual Studio 2022 (з встановленим .NET MAUI workload)
2. Обрати ціль запуску — Windows (WinUI)
3. Натиснути Run
